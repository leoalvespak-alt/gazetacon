# Identidade Visual - Gazeta dos Concursos

Este documento registra as cores e diretrizes visuais oficiais do projeto, conforme solicitado em 28/01/2026.

## Paleta de Cores (Brand Palette)

| Componente           | Cor (Hex) | Aplicação Sugerida                                            |
| :------------------- | :-------- | :------------------------------------------------------------ |
| **Vermelho Notícia** | `#C00000` | Barra de topo (G1), Logo (Veja), Títulos, Botões de destaque. |
| **Amarelo Alerta**   | `#FFD941` | Fundos de avisos, Botões secundários (Assine Agora), Ênfases. |
| **Preto Forte**      | `#000000` | Textos principais, Títulos em Light Mode, Headers.            |
| **Preto Fraco**      | `#313131` | Fundo principal em Dark Mode, Sidebar administrativa.         |
| **Branco**           | `#FFFFFF` | Fundo principal em Light Mode, Texto em fundos escuros.       |

## Diretrizes de Uso

1. **Hierarquia:** O vermelho `#9D0000` deve ser a cor dominante para atrair a atenção para informações críticas e identidade da marca.
2. **Alertas:** O amarelo `#FFD941` deve ser usado com moderação para destacar informações importantes sem a agressividade do vermelho.
3. **Contraste:** Sempre garantir que o texto branco `#FFFFFF` seja usado sobre o preto fraco `#313131` ou vermelho `#9D0000` para máxima legibilidade.
